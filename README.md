# House Price Prediction using Linear Regression

This project implements a **Linear Regression** model to predict house prices based on three important features: **area (square footage)**, **number of bedrooms**, and **number of bathrooms**. The dataset is preprocessed and divided into training and testing sets to evaluate the model’s performance. The trained model can estimate the price of a new house given its characteristics.

## Technologies Used

* Python
* Pandas
* Scikit-learn

## Features

* Data loading and preprocessing
* Model training using Linear Regression
* Performance evaluation using the R² score
* Price prediction for new houses

This project demonstrates the application of supervised machine learning to a real-world regression problem in the real estate domain.
